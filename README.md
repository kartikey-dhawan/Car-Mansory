# Car Mansory

Car Mansory is a simple web project where you can see different cars and interact with them.

## Features
- Fetches data from CarAPI (if it doesn’t work, it uses some sample data)
- Search cars by name or brand
- Filter cars by year or manufacturer
- Sort cars by name or year
- Add cars to favorites using heart button
- Favorites are saved in localStorage
- Dark mode and light mode option
- Responsive design with some hover effects

Note: I have used array methods like map(), filter() and sort() for search, filter and sorting.

## Technologies Used
- HTML
- CSS
- JavaScript
